# Programação Orientada a Objetos com Python

Este repositório contém os **códigos desenvolvidos em aula** na disciplina de **Programação Orientada a Objetos (POO)** com **Python**.  
Os exemplos aqui servem como referência prática dos conceitos abordados — herança, polimorfismo, encapsulamento, métodos mágicos, entre outros.

---

## Estrutura do Repositório

- Cada pasta representa um **módulo, tema ou aula**.
- Os códigos seguem as boas práticas de programação em Python (PEP 8).
- Alguns exemplos incluem **docstrings**, **tipagem**, e **testes unitários**.

---

## Tecnologias Utilizadas

- Python 3.10+
- Google Colab (para demonstrações)
- VS Code (para edição local)
- Git e GitHub (controle de versão)

---

2. **Abra o terminal** (ou PowerShell, no Windows).
