# Programação Orientada a Objetos com Python

Este repositório contém os **códigos desenvolvidos em aula** na disciplina de **Programação Orientada a Objetos (POO)** com **Python**.  
Os exemplos aqui servem como referência prática dos conceitos abordados — herança, polimorfismo, encapsulamento, métodos mágicos, entre outros.

---

## Estrutura do Repositório

- Cada pasta representa um **módulo, tema ou aula**.
- Os códigos seguem as boas práticas de programação em Python (PEP 8).
- Alguns exemplos incluem **docstrings**, **tipagem**, e **testes unitários**.

---

## Tecnologias Utilizadas

- Python 3.10+
- Google Colab (para demonstrações)
- VS Code (para edição local)
- Git e GitHub (controle de versão)

---

## Como Clonar este Repositório

Para obter uma cópia local deste repositório, siga os passos abaixo:

1. **Instale o Git** (se ainda não tiver instalado):  
   [https://git-scm.com/downloads](https://git-scm.com/downloads)

2. **Abra o terminal** (ou PowerShell, no Windows).

3. **Escolha uma pasta onde quer salvar o projeto** e execute:

    ```
   git clone https://github.com/joaolaoliveira/poo-nes-2025.git
    ```

4. Entre na pasta clonada:

    ```
    cd poo-nes-2025
    ```

5. (Opcional) Abra no VS Code:

    ```
    code .
    ```

## Criando seu Branch

Cada aluno deve criar um branch com seu próprio nome (sem espaços ou acentos em snake_case).
Exemplo:

   ```
   git checkout -b aluno_joaoluiz
   ```

Esse comando cria e já alterna para o seu branch.

## Salvando Suas Alterações

Após editar ou criar arquivos:

   ```
   git add .
   git commit -m "Adiciona exercícios da aula 03"
   git push origin aluno_joaoluiz
   ```
## Enviando suas Mudanças para o Professor (Pull Request)

Se quiseres contribuir com a main do repositório, você pode realizar um **Pull Request** para ser analisado e, possivelmente, aceito pelo professor

1. Acesse o repositório no GitHub.
2. Clique em **Compare & pull request**.
3. Verifique se:
   - O **base branch** é `main`.
   - O **compare branch** é o **seu branch** (`aluno_joaoluiz`, por exemplo).
4. Clique em **Create pull request**.
5. Aguarde a revisão e aprovação do professor.

⚠️ Os alunos não têm permissão para dar push direto na main. Todas as alterações devem passar por Pull Request.
